Gib Folgende URIs ein, um folgende Funktionen auszuführern:

http://localhost:9998/network/user/{id} -> Zeige Informationen eines Users an

http://localhost:9998/network/adduser/{username}/{vname}/{nname} -> Füge User hinzu

http://localhost:9998/network/userRemove/{user id} ->lösche User

http://localhost:9998/network/user/{id von user}/post/{id von Post} -> Zeige Posts von User

http://localhost:9998/network/posts -> zeige alle Posts

http://localhost:9998/network/users - zeige alle User

Die Informationen der user finden Sie in der users.txt
